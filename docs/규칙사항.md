# Git Convention
|구분|로그|
|:-:|:-:|
|기능 구현|feat|
|(오류) 수정|fix|
|삭제|delete|
|리펙터링|refactor|
|테스트|test|
|문서화|docs|
|긴급 수정|hotfix|
  
> branch는 위를 따라 **구분/이슈번호/세부설명** 형식으로 작성한다.
